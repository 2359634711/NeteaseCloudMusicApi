# NeteaseCloudMusicApi (C#, WebAPI)
## **非常感谢 [@AxHello](https://github.com/axhello) 提供 [PHP版本的](https://github.com/axhello/NeteaseCloudMusicApi) NeteaseCloudMusicApi**

搜集到以下可用API：

- 搜索API
- 歌曲详情API
- 歌词API
- 用户歌单API
- MV视频API

API返回的是json对象，前端可用js输出。
